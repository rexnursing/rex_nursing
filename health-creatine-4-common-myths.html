肌酸的 8 個最常見迷思：科學文獻幫你一一破解

RRex（護理師）整理｜資料來源：PubMed 同儕審查文獻｜2026 年更新

怕腎衰竭、怕禿頭、怕水腫……這些到底有多少是真的？

迷思 1：肌酸會造成長期水分滯留？

雖然有一些證據顯示肌酸補充會在短期內增加水分滯留，但也有數項研究顯示，在較長時間內，並不會改變總體水分。

📌出處：Antonio J 等人，2021，J Int Sports Nutr，PMID: 33557850

迷思 2：肌酸是合成代謝類固醇？

合成代謝類固醇是藥物，化學結構與肌酸不同，屬於管制物質。肌酸則與其他許多膳食補充品一樣，屬於膳食補充品範疇。

📌出處：Antonio J 等人，2021，J Int Sports Nutr，PMID: 33557850

迷思 3：肌酸會傷腎？

目前尚未發現肌酸補充與腎功能或肝功能惡化之間有強烈的關聯性證據，唯一有文獻記錄的副作用是體重增加。

📌出處：PMID: 15758854

一項孟德爾隨機化分析發現：關於肌酸補充可能觸發腎臟問題的疑慮，在健康個體中可能並不合理。

📌出處：PMC11232645

不過有一個個案報告：一名 18 歲健康男性在服用肌酸單水化合物進行健身時出現急性腎小管壞死，停止補充 25 天後完全恢復。2025 年另有一篇 17 歲青少年出現類似罕見不良反應（圓柱體腎病）的個案報告，顯示此類反應持續被記錄，但發生率極低。

📌出處：PMID: 25984094

迷思 4：副作用比安慰劑嚴重很多？

2025 年一項對 685 篇人類臨床試驗的全面性安全性分析結果顯示：安慰劑組有 13.2% 的研究報告副作用，肌酸組為 13.7%，兩者之間無顯著差異（p = 0.776）。

📌出處：Kreider RB 等人，2025，J Int Sports Nutr，PMID: 40198156（此論文發表費用由 Alzchem Group AG〔德國肌酸原料製造商〕資助，屬產業關聯，供讀者參考）

迷思 5：肌酸會讓人增加體脂肪？

目前並無充分證據顯示肌酸補充會增加體脂肪；體重增加主要來源為肌肉量和細胞內液的增加。

📌出處：Antonio J 等人，2021，J Int Sports Nutr，PMID: 33557850

迷思 6：一定要做「載入期」才有效？

在沒有載入期的情況下，每天 3–5 克的維持劑量可在較長時間後達到類似的肌肉肌酸飽和程度，雖然速度較慢。

📌出處：Antonio J 等人，2021，J Int Sports Nutr，PMID: 33557850

迷思 7：肌酸會造成脫水和肌肉痙攣？

現有證據並不支持肌酸補充會造成脫水或肌肉痙攣。

📌出處：Antonio J 等人，2021，J Int Sports Nutr，PMID: 33557850

迷思 8：肌酸會造成掉髮或禿頭？

雖然有一項研究顯示，2 週的載入方案後二氫睪固酮（DHT）水平有所上升，但目前並無臨床試驗直接確認肌酸補充會導致掉髮。

📌出處：Antonio J 等人，2021，J Int Sports Nutr，PMID: 33557850
